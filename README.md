  npm create vite@latest online-library -- --template react

  cd online-library

Step 2: Install Dependencies

  npm install react-router-dom @reduxjs/toolkit react-redux react-icons

Step 3: Download Project

Step 4: Run the Application

  npm run dev

The app will be available at http://localhost:5173
